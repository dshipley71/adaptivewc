# Example Data

<b><u>AHrefsTop</u></b>
* The top 10 news websites for every country listed on AHrefsTop was s
craped. If the request returned something other than a 200 status code
, a screenshot of the full page was taken and the HTML was saved.
    * <u>Captcha</u>: folder containing results from sites where the HTML
 contained captcha/cloudflare keywords 
    * <u>unknown</u>: folder containing results from sites where the HTML did not have captcha/cloudflare keywords

<b><u>Captcha Examples</u></b>
* <u>slider_examples.zip</u>: Folder containing slider captcha examples and t
he HTML associated with them, scraped from https://2captcha.com/demo/geetest-v4.
* <u>alphanumeric_lang-code.zip</u>: 100 alphanumeric examples of different languages captchas & associated HTML from captcha.com
